# Stored-Programs
.They represent named blocks of code stored directly within a database to automate tasks and manage data more effectively.
1. Procedures (Stored Procedures)Definition: A group of SQL statements compiled as a single unit to perform specific tasks.
   Invocation: Must be called explicitly by a user or an application using commands like EXECUTE or CALL.
2. Functions (User-Defined Functions)Definition: A stored program designed to perform a specific calculation or data transformation.
   Invocation: Called explicitly within other SQL queries or expressions, similar to built-in functions like SUM() or UPPER().
3. TriggersDefinition: A special type of stored procedure that is attached to a specific table or view.
   Invocation: Fired implicitly (automatically) by the database in response to specific events, such as INSERT,UPDATE, or DELETE operations.
